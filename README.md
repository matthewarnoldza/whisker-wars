
# Whisker Wars (starter)

A browser-based, mobile-friendly cat-collecting battler built with **Vite + React + TypeScript + Tailwind + Zustand + Framer Motion**.

## Quick start

```bash
npm i
npm run dev
```

Build:
```bash
npm run build
npm run preview
```

## Notes
- Autosaves to localStorage every ~1.5s.
- Buy bait, use it to befriend cats, select 3 in **Collection**, then fight in **Battle Arena**.
- Dice rolls animate; abilities apply lightweight effects for quick, fun battles.
